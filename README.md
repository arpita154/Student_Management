- npm or yarn

1. **Install dependencies:**

2. **Start the development server:**
   ```bash

    ```bash

🛠️ Usage Guide
<details> <summary><b>Click to view detailed usage instructions</b></summary>
Adding a Student
Click the "Add Student" button
Fill in the student details (Name, ID, Email, etc.)
Click "Submit" to add the student
Viewing Students
All students are displayed in the main list
Scroll through to see all records
Deleting a Student
Find the student in the list
Click the "Delete" button next to their name
Confirm the deletion
Filtering & Searching
Use the search bar to find students by name or ID
Sort by clicking on column headers
</details>
